Unity Peek

About Unity Peek

Unity Peek is a tool that allows you to view the built hierarchy and inspector views of Unity games using the UnityPeekPlugin BepInEx plugin and the UnityPeek software. It provides a convenient way to inspect and interact with the game objects and components in real-time.

With Unity Peek, you can easily explore the structure of Unity games, view their properties, and make modifications on the fly. This can be particularly useful for debugging, testing, or simply gaining insights into how the game is built.

Unity Peek currently works with most Unity versions but requires the game to be built with Mono and not IL2CPP since it requires a BepInEx plugin to also be installed in the targeted game. Comunication to the plugin and the software is through a configurable local ip address using a socket.

Unity Peek is currently in development by me and currently can only veiw the hierachy of the game and transform data of each object but the ability to change transform values, see other components and much more is coming soon. For more details please checkout the Github linked below where you can find installation instructions, build instructions and more info!

Features

  • View full GameObject Hierarchy
  • View of Transform Data
  • Compatible with most Unity builds
  • Built using BepInEx 6.0.0 plugin system and Avalonia
